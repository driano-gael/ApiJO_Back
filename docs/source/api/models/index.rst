Modèles Api
===========

Les modèles définissent les structures de données pour les entités des Jeux Olympiques.

- **Discipline** : Sports olympiques avec icônes
- **Lieu** : Sites sportifs pour les événements
- **Evenement** : Sessions avec gestion des places
- **Epreuve** : Compétitions dans les disciplines
- **Offre** : Packages commerciaux
