Infrastructure Payment
======================

Le module infrastructure contient les composants d'infrastructure pour l'intégration
avec les passerelles de paiement externes.

.. toctree::
   :maxdepth: 2
   :caption: Composants d'infrastructure:

   payment_gateway

**Description générale**

Ce module fournit les composants nécessaires pour l'intégration avec les
services de paiement externes, incluant des interfaces mock pour les tests
et le développement.
