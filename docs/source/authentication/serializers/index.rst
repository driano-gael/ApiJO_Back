Sérialiseurs d'Authentification
===============================

Les sérialiseurs gèrent l'inscription des utilisateurs et l'authentification JWT avec validation avancée.

# Fonctionnalités


- **Inscription sécurisée** avec validation complète des données
- **Authentification JWT** avec informations de rôle enrichies
- **Validation métier** : emails, mots de passe, données personnelles
- **Gestion des rôles** : client, employé, admin
